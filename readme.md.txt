 X Academy - QA Automation (Clase 5)

Proyecto de automatización QA realizado en Cypress sobre la plataforma SauceDemo como parte de la formación en QA Automation de X Academy.

 Tecnologías utilizadas

* Cypress
* JavaScript
* Git
* GitHub
* Trello (Bug Reporting)

## Casos automatizados

### Login

* Login exitoso con credenciales válidas
* Validación de acceso al inventario

### Inventario

* Agregar productos al carrito
* Agregar múltiples productos y validar contador
* Validación de comportamiento del inventario

### Carrito

* Eliminar producto desde el carrito

### Checkout

* Checkout con datos válidos
* Checkout con campos obligatorios vacíos
* Validación de mensajes de error
